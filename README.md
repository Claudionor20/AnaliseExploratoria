# Análises com base no arquivo base_trabalho
Repositório criado para a aula de Prática Estatística 1

## Resumo de algumas análises exploratórias

### Histograma da variável idade

Foi possível verificar que a idade dos detentos está concentrada entre 50 e 60 anos. Entretanto, a base está bem variável possuindo muitos individuos de todas as cidades.

### Boxplot de tempo preso

Com relação ao tempo que o preso encontra-se detido, foi identificado que os valores mais frequentes são entre 50 e 70 meses, possuindo um outlier acima de 85 meses.

<img width="2400" height="1800" alt="boxplot_tempo_preso" src="https://github.com/user-attachments/assets/1f56454a-288f-4238-8deb-dad4610824ea" />

### Boxplot de score periculosidade

É possível identificar que os maiores scores de periculosidade, ou seja, os detentos mais perigosos estão concentrados com Fundamental e Medio completo, enquanto os de ensino Superior possuem escores levemente menores.
